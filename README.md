# Phonetic-Archive

User Guide

1. Open the project

Open the main project link on a computer browser:

https://phonetic-archive.onrender.com/

Open the mobile controller link on an iPhone Safari browser:

https://phonetic-archive.onrender.com/controller

The computer screen is used for the main visual interface, and the phone is used as a motion controller.

2. Enable phone motion

On the phone controller page, tap:

ENABLE MOTION

Allow motion and orientation permission when the browser asks. After this, the phone will start sending tilt and shake data to the computer screen.

3. Type letters and words

On the computer screen, type English letters or words using the keyboard. Each letter will appear as an abstract visual form with different colours, materials, distortions, and movement behaviours.

4. Control the image with the phone

Tilt or shake the phone to influence the movement of the letter-based image. The image will respond to the phone’s direction and motion, creating a more unstable and bodily visual composition.

5. Save a word into the archive

After typing a word, press:

Enter

The current word composition will be saved into the archive page as a small visual image.

6. View the archive

The archive page displays all saved word-images as a visual catalogue. The phone controller can still influence the movement inside the archived image windows.

7. Open an archive detail

Click one of the archived image windows to open it in a larger detail view. Click again or press:

Esc

to return to the archive page.

8. Other controls

Press:

Backspace

to delete the last typed letter.

Press:

8

to enter or exit fullscreen mode.
